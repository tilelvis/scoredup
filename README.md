# ScoreUp — AI Neynar Score Booster

## Features
- Live Farcaster trends
- Grok-3 AI tasks (unique per user)
- $SCORE ERC-20 credits (Base)
- Anti-spam deduplication

## Setup
1. Deploy `contracts/ScoreToken.sol` → Base
2. Update `.env` with keys + TOKEN_ADDRESS
3. `cd backend && npm i && node server.js`
4. `cd miniapp && npm i && npm run dev`
